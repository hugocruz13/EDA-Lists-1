\hypertarget{index_autotoc_md4}{}\doxysubsection{\texorpdfstring{Motivação}{Motivação}}\label{index_autotoc_md4}
Este projeto individual tem como objetivo reforçar e aplicar os conhecimentos adquiridos na Unidade Curricular de Estruturas de Dados Avançadas (EDA), especialmente no que diz respeito à manipulação de estruturas de dados dinâmicas na linguagem de programação C.

\DoxyHorRuler{0}
\hypertarget{index_autotoc_md6}{}\doxysubsection{\texorpdfstring{Objetivo}{Objetivo}}\label{index_autotoc_md6}
Desenvolver soluções de software para manipulação de listas ligadas representando matrizes de inteiros, aplicando os conhecimentos adquiridos sobre estruturas de dados dinâmicas, armazenamento em ficheiro, modularização e documentação com Doxygen.

\DoxyHorRuler{0}
\hypertarget{index_autotoc_md8}{}\doxysubsection{\texorpdfstring{Fase 1 -\/ Listas Ligadas}{Fase 1 -\/ Listas Ligadas}}\label{index_autotoc_md8}
\hypertarget{index_autotoc_md9}{}\doxysubsubsection{\texorpdfstring{Descrição do Problema}{Descrição do Problema}}\label{index_autotoc_md9}
Implementar uma solução para calcular a soma máxima possível dos inteiros de uma matriz, de modo que nenhum dos inteiros selecionados compartilhe a mesma linha ou coluna.\hypertarget{index_autotoc_md10}{}\doxysubsubsection{\texorpdfstring{Funcionalidades a Implementar}{Funcionalidades a Implementar}}\label{index_autotoc_md10}

\begin{DoxyEnumerate}
\item {\bfseries{Estrutura de Dados Dinâmica}}\+: Utilizar listas ligadas para representar a matriz de inteiros.
\item {\bfseries{Carregamento de Dados}}\+: Carregar a matriz a partir de um ficheiro de texto, considerando qualquer dimensão, com os valores separados por vírgulas.
\item {\bfseries{Alteração de Valores}}\+: Permitir a alteração dos inteiros na estrutura de dados.
\item {\bfseries{Inserção de Linhas/\+Colunas}}\+: Adicionar novas linhas ou colunas na matriz.
\item {\bfseries{Remoção de Linhas/\+Colunas}}\+: Eliminar linhas ou colunas existentes.
\item {\bfseries{Listagem Tabular}}\+: Exibir todos os inteiros da matriz de forma tabular na consola.
\item {\bfseries{Cálculo da Soma Máxima}}\+: Determinar a soma máxima possível dos inteiros, respeitando a condição de exclusividade de linha e coluna.
\end{DoxyEnumerate}
